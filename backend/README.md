# Backend Setup

## Directory Structure

```
backend/
│
├── bin/
│   └── www.js
│
├── models/
│   ├── author.js
│   ├── book.js
│   ├── bookinstance.js        ← You may remove BookInstance if not used this round
│   └── genre.js
│
├── routes/
│   ├── author.js
│   ├── book.js
│   └── genre.js
│
├── app.js
├── package.json
├── package-lock.json
├── populate.js
└── README.md
```

## Install Dependencies

```
npm install
```


## IMPORTANT

```
Make sure the Backend server port has Visibility Public
```

## Start Backend Server

```
npm start
```
